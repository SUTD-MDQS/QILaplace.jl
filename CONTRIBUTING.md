## How to contribute

Pull requests for refactors, documentation, performance improvements, or new ideas are very welcome. If you make the project better, you are a contributor. Here's a quick guide on how to get started:

1. **Fork the repository** and create your branch from `master`.
2. **Implement your changes**. If you're adding a feature, please add tests.
3. **Run the tests** locally using `julia --project=. test/runtests.jl`.
4. **Submit a pull request**. 

We appreciate any help, whether it's fixing a typo or implementing a new tensor-network algorithm!
